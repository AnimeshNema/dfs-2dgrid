Initializes a 2D grid, based on user input and performs Depth First Search algorithm to find a path, if it exists.<br/>

**#Build Instructions<br/>**
Clone the repository<br/>
**Open Terminal<br/>**
	cd dfs-2dgrid<br/>
	mkdir build<br/>
	cd build<br/>
	cmake ..<br/>
	make<br/>
	./gridSearchDFS<br/>
